Privacy Policy

FileRenamer Chrome Extension  ·  Last updated: June 2026

Summary: FileRenamer processes all files entirely on your device. No data is collected, transmitted, or stored on any server. Your files, names, and IDs never leave your computer.

1. What FileRenamer Does

FileRenamer is a Chrome extension that helps you batch rename files by matching filenames against a local roster (Name & ID spreadsheet) and applying a configurable naming template.

2. Data We Do NOT Collect

FileRenamer does not collect, transmit, store, or share any of the following:

  • File names or file contents
  • Names or IDs from your roster spreadsheet
  • Renaming templates or fixed text you enter
  • Usage statistics or analytics
  • Any personally identifiable information

3. How Your Data is Processed

All processing takes place locally in your browser using the File System Access API:

  • Files are read from your selected folder using browser-granted permissions
  • Roster data (names and IDs) is loaded from your spreadsheet into browser memory only
  • Matching is performed in JavaScript in the browser tab
  • Renamed copies are written back to your local file system
  • Nothing is sent to any external server at any point

4. Permissions Used

FileRenamer uses one Chrome permission:

  • tabs — to open the FileRenamer interface in a full browser tab

File access is granted by you through the browser's native file picker. This permission is session-only and not stored by the extension.

5. Third-Party Services

FileRenamer does not use any third-party analytics, tracking, or cloud services. All libraries (SheetJS for Excel parsing, PDF.js for PDF rendering) run entirely in your browser.

6. Children's Privacy

FileRenamer does not collect any personal information from anyone, including children under 13.
